Python Day Maracay
O conocido también como PyDay es un evento gratuito organizado por la comunidad de desarrolladores Python con el objetivo de difundir el uso del lenguaje dentro de todos los entornos de la comunidad venezolana. Python es un lenguaje de programación de código abierto de alto nivel, multipropósito, multiplataforma y multiparadigma fácil de aprender y divertido al usar.
